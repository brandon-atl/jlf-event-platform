## Quick Summary
- Custom Event Management ERP for Just Love Forest (justloveforest.com)
- Single-system architecture: Custom form → Stripe Checkout → PostgreSQL
- **No Acuity integration** — Acuity is removed from the booking flow
- Backend: FastAPI + SQLAlchemy + Alembic
- Frontend: React (Vite) — dashboard mockup exists at `reference/jlf-erp-final.jsx`
- Registration states: PENDING_PAYMENT → COMPLETE → CANCELLED/REFUNDED (or EXPIRED)
